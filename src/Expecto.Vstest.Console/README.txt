
You can debug the test adapter by setting this project to 

 * Start Action / Start external program = C:\Program Files (x86)\Microsoft Visual Studio\2017\{VisualStudio Version Type}\Common7\IDE\CommonExtensions\Microsoft\TestWindow\vstest.console.exe
 * Start Options / Command line arguments = Expecto.Vstest.Console.exe /testadapterpath:.

 Then start the debugger (e.g. F5).
